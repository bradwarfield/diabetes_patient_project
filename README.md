# diabetes_patient_project
Project to answer 9 problems from a healthcare dataset. Tools used SQL, Tableau, Desmos.

The write up for this project can be seen at https://warfield-analyst.carrd.co/#sqldiabetes. Go check it out!

For more on my journey from a data-driven teacher to a curiosity-driven analyst, connect with me at https://www.linkedin.com/in/brad-warfield/. Questions and comments are most welcome.

The problems to solve for this project came from the Data Analytics Accelerator program from Data Career Jumpstart.
