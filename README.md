# diabetes_patient_project
Project to answer 9 problems from a healthcare dataset. Tools used SQL, Tableau, Desmos. Project report can be found at https://warfield-analyst.carrd.co/#sqldiabetes

Folder contains SQL created by Brad Warfield, MSIS. I created them using MySQL Workbench 8.0

Folder contains all images for this data project. All of the SQL queries have an image made with Carbon ttps://carbon.now.sh. Other images include screenshots of tables and graphs.
